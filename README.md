# automatically_separating_files
This System can rearrange the files or create files of same data like photoes, pdf, dox, word etc are in the different files.
This is an automated system.
You just need to Download the python code.
Run the code in your terminal.
It asks you the Directory where you want to perform the rearrangement.
Provide the Directory.
It will automatically makes files like all photoes, pdf, word files etc are in a seperate files.
